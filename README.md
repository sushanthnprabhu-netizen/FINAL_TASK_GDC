# Whispers in Hall — Functional Horror Menu Prototype

A psychological horror-themed Unity UI prototype created for the IITK Game Dev Club recruitment task.

This project focuses on:
- multi-scene UI navigation
- persistent background audio
- volume control system
- atmospheric horror presentation
- clean Unity UI setup

---

# Features

## Main Menu
- Custom horror-themed background
- Interactive START, OPTIONS, and QUIT buttons
- Subtle hover audio feedback
- Atmospheric ambient music

- START loads the Game Scene
- OPTIONS loads the Options Scene
